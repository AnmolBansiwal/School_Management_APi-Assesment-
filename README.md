# School Management API

A Node.js, Express.js, and MySQL based API project for managing school data.

## Features
- Add new schools
- List schools sorted by proximity to user location

## Tech Stack
- Node.js
- Express.js
- MySQL
- Postman

## Setup Instructions

1. Clone the repository
2. Run `npm install`
3. Create `.env` file
4. Add MySQL credentials
5. Create database and table
6. Run `npm run dev`

## Environment Variables

```env
PORT=5000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_mysql_password
DB_NAME=school_management
